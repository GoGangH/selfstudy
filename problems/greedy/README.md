# Greedy (그리디)

## 개념
그리디 알고리즘은 매 선택에서 현재 시점에 가장 최적인 답을 선택하는 알고리즘입니다.
항상 최적해를 보장하지는 않지만, 특정 조건을 만족하면 최적해를 구할 수 있습니다.

## 적용 조건
1. **탐욕적 선택 속성(Greedy Choice Property)**: 각 단계에서 최선의 선택이 전체 최적해로 이어짐
2. **최적 부분 구조(Optimal Substructure)**: 문제의 최적해가 부분 문제의 최적해로 구성됨

## 문제 해결 전략
1. 선택 절차: 현재 상태에서 최선의 선택
2. 적절성 검사: 선택이 문제의 조건을 만족하는지 확인
3. 해답 검사: 문제가 해결되었는지 확인

## 대표 문제 유형
- 거스름돈 문제
- 활동 선택 문제
- 최소 신장 트리 (크루스칼, 프림)
- 허프만 코딩
- 작업 스케줄링
