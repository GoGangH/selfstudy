# BFS/DFS (너비우선탐색/깊이우선탐색)

## 개념
그래프나 트리를 탐색하는 기본적인 알고리즘입니다.

### BFS (Breadth-First Search)
- 시작 정점에서 가까운 정점부터 탐색
- **큐(Queue)** 자료구조 사용
- 최단 경로 탐색에 유용
- 시간복잡도: O(V + E)

### DFS (Depth-First Search)
- 한 방향으로 끝까지 탐색한 후 다른 방향 탐색
- **스택(Stack)** 또는 **재귀** 사용
- 경로 탐색, 사이클 검사에 유용
- 시간복잡도: O(V + E)

## BFS vs DFS 선택 기준

| 구분 | BFS | DFS |
|------|-----|-----|
| 자료구조 | 큐 | 스택/재귀 |
| 최단 경로 | ⭕ 가능 | ❌ 불가능 |
| 메모리 | 많이 사용 | 적게 사용 |
| 용도 | 최단거리, 레벨 탐색 | 경로 존재 여부, 사이클 |

## 대표 문제 유형
- **BFS**: 최단 거리, 미로 탐색, 토마토 문제
- **DFS**: 연결 요소 개수, 단절점/단절선, 위상 정렬

## 구현 팁
- 방문 체크 배열 필수
- 좌표 이동 시 dx, dy 배열 활용
- 큐는 LinkedList, 스택은 재귀나 Stack 사용
