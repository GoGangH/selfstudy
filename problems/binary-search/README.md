# Binary Search (이분탐색)

## 개념
이분탐색은 정렬된 배열에서 특정 값을 찾는 알고리즘입니다.
탐색 범위를 절반씩 줄여나가므로 O(log n)의 시간복잡도를 가집니다.

## 동작 원리
1. 배열의 중간 값을 선택
2. 중간 값과 찾으려는 값을 비교
3. 찾으려는 값이 작으면 왼쪽 절반, 크면 오른쪽 절반을 탐색
4. 값을 찾거나 탐색 범위가 없어질 때까지 반복

## 구현 방법
- **반복문 (Iterative)**: 스택 오버플로우 위험 없음
- **재귀 (Recursive)**: 코드가 직관적

## 주의사항
- 배열이 반드시 정렬되어 있어야 함
- `mid = (left + right) / 2` 대신 `mid = left + (right - left) / 2` 사용 (오버플로우 방지)
- lower_bound, upper_bound 구현에 유의

## 응용
- **파라메트릭 서치**: 최적화 문제를 결정 문제로 변환
- **Lower Bound**: 찾으려는 값 이상이 처음 나타나는 위치
- **Upper Bound**: 찾으려는 값을 초과하는 첫 위치
