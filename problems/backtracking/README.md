# Backtracking (백트래킹)

## 개념
백트래킹은 가능한 모든 경우의 수를 탐색하되, 조건을 만족하지 않으면
이전 단계로 돌아가서(backtrack) 다른 경로를 탐색하는 알고리즘입니다.

## 핵심 아이디어
- 모든 경우의 수를 탐색 (완전 탐색)
- **가지치기(Pruning)**: 조건을 만족하지 않으면 더 이상 진행하지 않음
- DFS 기반의 재귀적 구현

## 백트래킹 vs DFS
- **DFS**: 모든 경로를 끝까지 탐색
- **백트래킹**: 조건을 만족하지 않으면 더 이상 탐색하지 않음 (가지치기)

## 구현 패턴

```java
void backtrack(int depth, 현재상태) {
    // 1. 종료 조건 (목표 도달)
    if (depth == target) {
        // 결과 저장
        return;
    }

    // 2. 모든 선택지 탐색
    for (int i = 0; i < choices; i++) {
        // 3. 유효성 검사 (가지치기)
        if (!isValid(i)) continue;

        // 4. 선택
        makeChoice(i);

        // 5. 재귀 호출
        backtrack(depth + 1, 다음상태);

        // 6. 선택 취소 (백트래킹)
        undoChoice(i);
    }
}
```

## 대표 문제 유형
- **N-Queen 문제**: N×N 체스판에 N개의 퀸 배치
- **순열/조합 생성**: 모든 경우의 수 나열
- **부분집합의 합**: 특정 합을 만드는 부분집합 찾기
- **스도쿠**: 9×9 스도쿠 퍼즐 풀이
- **미로 탐색**: 모든 가능한 경로 찾기
- **단어 퍼즐**: 단어 찾기, 크로스워드

## 최적화 기법
1. **조기 종료**: 가능한 빨리 불가능한 경로 판단
2. **정렬**: 가능성 높은 선택지를 먼저 탐색
3. **비트마스킹**: 방문 체크를 비트 연산으로 처리
4. **메모이제이션**: 중복 계산 방지

## 시간복잡도
- 최악의 경우: O(선택지^깊이)
- 가지치기로 실제 수행 시간은 훨씬 단축됨
